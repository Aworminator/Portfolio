Projects

Alec Garnica Audio Portfolio

This audio portfolio website showcases my work in sound design and music production, featuring a custom-built music player created with Vue.js. The player manages playback state, ensures only one track plays at a time, and provides real-time progress updates and click-to-seek functionality for a seamless listening experience. I leveraged Vue's reactivity and component system to synchronize UI elements with audio events, while custom CSS delivers a modern, responsive layout that adapts beautifully to any device. The result is an interactive, polished site where visitors can easily explore and enjoy my audio projects.
