About Me

Hi, I'm Alec Garnica — a developer passionate about building creative, interactive experiences with clean code and thoughtful design. My work includes an audio portfolio built with Vue.js, featuring a custom music player and seamless user experience, as well as a responsive doggy daycare website crafted in React for a real client. I also build robust backend solutions using Node.js and Express, powering full-stack apps like my inventory POS system with secure, scalable server logic. I enjoy taking projects from concept to launch, using JavaScript, modern frameworks, and CSS to deliver polished, intuitive results. I'm always learning, experimenting, and striving to create web apps that are both functional and engaging. Collaboration, quality, and making technology approachable drive my work — and I'm excited to keep growing and building.
